# bopnet.github.io

Project site for **Bopwire** — a peer-to-peer music network where the listeners
are the infrastructure, content is verified by acoustic fingerprint, and every play
settles on a purpose-built blockchain.

- **Site:** https://bopnet.github.io
- **White paper:** [whitepaper.html](whitepaper.html) · [PDF](whitepaper.pdf) · [source](WHITEPAPER.md)
- **Source code:** https://github.com/knobcore/bopwire

This site is static (plain HTML/CSS, no build step). `.nojekyll` keeps GitHub Pages
from post-processing the files.
